Folder "Offers_program" contains the program's code.

Folder "Database" contains database that must be copied anywhere on computer or local network
At first start of program it is necessary to first go to 'Settings' and set database location
so that program could read and save data from database.

Program was built using following technologies:
1. Java SE8;
2. JavaFX 8;
3. Scene Builder;
4. Java Database (Apache Derby);
5. Apache POI.
